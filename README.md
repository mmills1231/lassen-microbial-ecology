This study examined 16S and 18S rRNA microbial community composition across geothermal environments within Lassen Volcanic National Park, including acid-sulfate and chloride-rich sites. Analyses focused on microbial diversity, composition, and environmental relationships. 

Analysis included using phyloseq objects, rarefaction, relative abundance analyses, diversity analyses, NMDS, PERMANOVA, and network analyses.

All of the analyses were conducted in R using the following packages:
- phyloseq
- vegan
- tidyverse
- ggplot2
- pairwiseAdonis
