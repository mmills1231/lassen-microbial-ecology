# FAPROTAX workflow

This protocol describes the workflow for assigning predicted microbial functions to the Lassen 16S dataset using FAPROTAX v1.2.12.

## 1. Navigate to the FAPROTAX directory

```bash
cd ~/FAPROTAX/faprotax_lassen/FAPROTAX_1.2.12
```

## 2. Confirm the Python version

```bash
python3.14 --version
```

Expected output:

```bash
Python 3.14.6
```

## 3. Install required packages

```bash
python3.14 -m pip install numpy pandas
```

## 4. Prepare the input table

The input table (`FAPROTAX_input_16S.txt`) was generated by extracting the ASV count data and corresponding SILVA taxonomy assignments from the rarefied 16S phyloseq object (`rare_16S`), then combining them into the tabular format recommended for FAPROTAX. The resulting table contains ASV counts for each sample and a final `taxonomy` column formatted for FAPROTAX.

Input file:

`FAPROTAX_input_16S.txt`

Example:

```text
#OTU ID    Sample1    Sample2    Sample3    taxonomy
ASV16S_0001    5247    2702    0    Archaea;Crenarchaeota;Thermoprotei;Sulfolobales;Sulfolobaceae;NA;NA
ASV16S_0002    15      4       0    Bacteria;Proteobacteria;Gammaproteobacteria;...
```

## 5. Run FAPROTAX

```bash
python3.14 collapse_table.py \
-i FAPROTAX_input_16S.txt \
-o functional_table_lassen.tsv \
-g FAPROTAX.txt \
-c "#" \
-d taxonomy \
--omit_columns 0 \
--column_names_are_in last_comment_line \
--omit_unrepresented_groups \
-r report_lassen.txt \
-n columns_after_collapsing \
-s subtables_lassen \
--out_groups2records_table groups2records_lassen.txt \
--out_group_overlaps overlaps_lassen.txt \
--out_group_definitions_used annotations_lassen.txt \
-v
```

## Output files

* `functional_table_lassen.tsv`
* `report_lassen.txt`
* `groups2records_lassen.txt`
* `overlaps_lassen.txt`
* `annotations_lassen.txt`
